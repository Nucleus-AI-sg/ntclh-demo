# PRD-06: Analytics & Reporting

## Overview

The real-time analytics dashboard and compliance reporting page. Demonstrates adaptive analytics for programme performance, visual KPI dashboards for leadership, and automated government report generation for SSG, WSG, and IMDA. Directly addresses the challenge brief's "Automated Compliance Reporting" use case and the metric of 100% programme visibility.

## Route

`/analytics`

## Proposal Mapping

- Feature 5: Real-Time Analytics, Reporting & Compliance Automation
- Subsections: Adaptive Analytics Engine, Automated Government Reporting, Data Governance & Visual Dashboards

## User Stories

- As a **programme leader**, I want to view placement rates, efficiency metrics, and trends at a glance so I can make strategic decisions.
- As a **coordinator**, I want the system to generate compliance reports automatically so I avoid manual Excel compilation.
- As a **coordinator**, I want to spot emerging risks to placement targets before they become entrenched.
- As a **programme leader**, I want to compare performance across programmes, cohorts, and sectors so I can allocate resources effectively.

Three top-level tabs:
- **Performance Overview** (default)
- **Programme Deep Dive**
- **Compliance Reports**

### Tab 3: Compliance Reports

#### 3a. Report Templates

A list of available report types:
- SSG Training Grant Claim Submission
- WSG Monthly Placement Outcome Return
- IMDA Programme Performance Summary
- Internal Quarterly Review

Each shows: last generated date, next due date, status (ready / pending data / overdue).

#### 3b. Report Generator

When a report type is selected:

**Report Configuration**:
- Reporting period selector (month/quarter)
- Programme scope (all or specific)
- "Generate Report" button

**Pre-Generation Checks**:
- Data completeness indicator: "96% of records complete"
- Missing data flags: "4 trainees missing verification status"
- Discrepancy warnings: "1 employer name mismatch detected"

**Generated Report Preview**:
- Formatted table matching the government agency's prescribed template
- Summary statistics
- Audit trail section showing data sources and verification timestamps
- "Download as Excel" and "Download as PDF" buttons

#### 3c. Report History

Table of previously generated reports:
- Report type, period, generated date, generated by, status (submitted / draft)
- Download link for each

## Interactions

1. **Tab switching**: Loads the selected analytics view
2. **KPI card click**: Expands to show detailed breakdown
3. **Chart hover**: Tooltips with exact values
4. **Programme selector**: Filters all deep-dive charts
5. **Generate report**: Shows loading animation, then renders the report preview
6. **Download buttons**: Triggers browser download of mock Excel/PDF file
7. **At-risk alert click**: Navigates to the relevant section (e.g., matching page for unfilled vacancies)
8. **Funnel stage click**: Drills down to show trainees at that stage
